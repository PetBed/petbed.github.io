🌐 Connect Google Sheet

Dynamically import student score records directly from any shared Google Spreadsheet. Updates publish instantly when you sync!

Setup Instructions:
  1. Create a Google Spreadsheet.
  2. Rename your second tab to exactly AR 1.
  3. Set score column headers on Row 2 of that tab. The first column must be Name. Subsequent columns should match subject tags exactly (e.g. BM, BI, MATE, PSV, PJK).
  4. Click Share in Google Sheets, and change access to "Anyone with the link can view".
  5. Copy your browser's address URL, paste it above, or save it as the DEFAULT_SHEET_ID inside the HTML code!
