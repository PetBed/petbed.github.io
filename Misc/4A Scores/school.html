🌐 Connect School Google Sheet

Dynamically import records for all grades and classrooms. The system instantly groups and maps your students, performance aggregates, and subject averages.

Spreadsheet Format Requirements:
  1. Set up a sheet tab named exactly School Scores.
  2. Row 1 must contain headers configured in the following positional order:
    • Column 1: Class (e.g., 4A, 5A, 4B, 5B)
    • Column 2: Name (Student's Name)
    • Subsequent Columns: Dynamic Subjects (e.g., BM, BI, MATE)
  3. The system automatically computes the student cohort (e.g., Form 4 or Form 5) from the first character of the Class column value!
  4. Set share permissions on Google Sheets to "Anyone with the link can view" before testing.
